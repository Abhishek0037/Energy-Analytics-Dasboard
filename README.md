#  Energy Analytics Dashboard

##  Overview

The Energy Analytics Dashboard is a data analytics project developed using Python, Pandas, and Power BI to analyze household energy consumption patterns. The project focuses on data cleaning, preprocessing, exploratory analysis, and dashboard visualization to identify energy usage trends and generate actionable insights.

---

##  Objectives

- Clean and preprocess raw energy consumption data.
- Handle missing and invalid values.
- Analyze hourly and monthly electricity usage patterns.
- Identify peak energy consumption periods.
- Build an interactive Power BI dashboard for visualization.

---

##  Technologies Used

- Python
- Pandas
- Power BI
- Matplotlib



---

##  Data Preprocessing

The following preprocessing steps were performed:

- Combined Date and Time into a Datetime column.
- Converted numerical columns into appropriate data types.
- Removed invalid and missing values.
- Generated a cleaned dataset for analysis.


---

##  Dataset

The original dataset contained over 1 million records. Due to GitHub size limitations, a representative sample dataset has been included in this repository for demonstration purposes.

---

##  Analysis Performed

### Statistical Analysis

- Average Power Consumption
- Maximum Power Consumption
- Minimum Power Consumption

### Time-Based Analysis

- Hourly Energy Consumption
- Monthly Energy Consumption
- Peak Demand Identification

---

##  Key Findings

- Average Consumption: 1.108 kW
- Maximum Consumption: 10.67 kW
- Peak Consumption Hour: 9 PM
- Highest Consumption Month: December
- Lowest Consumption Month: August

---

##  Dashboard Features

- KPI Cards
- Hourly Consumption Trend Analysis
- Monthly Consumption Trend Analysis
- Key Insights Section

---

##  Insights

- Household energy demand is highest during evening hours.
- Peak usage occurs around 9 PM.
- Winter months show significantly higher consumption.
- December records the highest average energy usage.
- August records the lowest average energy usage.
